# Saldo

Saldo is a comprehensive personal finance dashboard designed for the next generation of financially independent college students. Built exclusively for iOS 26+, the application provides a modern, secure, and intuitive platform for managing allowances, savings, and recurring expenses.

## The Vision

The journey to financial autonomy is a critical phase of life that is often underserved by formal education. Saldo was conceived as a solution for college students navigating the challenges of living from one allowance to the next. By bridging the gap between receiving an allowance and achieving true financial independence, Saldo empowers users to gain complete control over their financial health through transparency and actionable insights.

## Achievement

Saldo was recognized as one of the premier project entries in the **Apple Swift Student Challenge 2026**, where it was selected as a winning submission.

## Core Features

Saldo is engineered using only native Apple frameworks, ensuring high performance, privacy, and full integration with the iOS ecosystem—without any external dependencies.

### Visual Savings (Grails)
The application introduces a unique "Grails" system for tracking long-term savings goals. Using Apple's **Vision framework**, Saldo automatically extracts the foreground of user-uploaded images to create masked, high-definition previews that keep savings objectives prominently visible and motivating.

### Intelligent Receipt Processing
Expense tracking is streamlined through advanced document scanning. Leveraging the native iOS 26 `RecognizeDocumentsRequest`, the application processes physical receipts to extract transaction metadata and amounts. The system employs a multi-strategy extraction algorithm to filter out promotional content and accurately record expenses.

### Adaptive Financial Theming
The user interface adapts dynamically to the user's financial status. Based on the current balance and allowance cycles, the app smoothly transitions through a curated theme system:
- **Alert**: Red color schemes for critical balance levels.
- **Moderate**: Amber/Yellow tones for cautious spending ranges.
- **Stable**: Green accents indicating a healthy financial state.

### Subscription & Recurring Payment Management
Users can maintain a consolidated view of all recurring commitments, including streaming services, AI tools, and educational subscriptions. The system provides clear categorization and tracking to prevent unforeseen impacts on the user's balance.

### Comprehensive Transaction History
A detailed transaction ledger tracks every manual entry, receipt scan, and recurring payment. Users can review histories, view weekly spending trends via integrated data visualizations, and access automated financial insights.

### Modern Interaction Design
Saldo features a premium software aesthetic built on modern SwiftUI principles:
- **Liquid Glass**: A custom design system incorporating high-fidelity glassmorphism and blur effects.
- **Gesture-Driven UX**: Fluid navigation utilizing multi-detent bottom sheets and custom scroll-synchronized animations.
- **Strict Concurrency**: Fully optimized for Swift 6, ensuring thread safety and responsive performance across all data processing tasks.
